6. Set Up PyPI (Optional)

If you want to publish to PyPI:

  1. Create PyPI account (if you don't have one)

  2. Go to https://pypi.org/account/register/

  3. Create API token

  4. Go to https://pypi.org/manage/account/token/

  5. Create a token for the pyZ3 project

  6. Configure GitHub secrets

  7. Go to your repository ‚Üí Settings ‚Üí Secrets and variables ‚Üí Actions

  8. Add PYPI_API_TOKEN secret with your token

  9. Enable trusted publishing (recommended)

  10. Configure at https://pypi.org/manage/account/publishing/
