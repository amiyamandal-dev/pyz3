)

API Reference

pyz3.wheel.Platform

Enum of supported platforms with properties: - zig_target - Zig target triple - wheel_platform - Wheel platform tag - current() - Detect current platform

pyz3.wheel.BuildConfig

Configuration for building wheels: - target_platform: Platform - Target platform - optimize: str - Optimization level - python_version: str - Python version - output_dir: Path - Output directory

pyz3.wheel.WheelBuilder

Main wheel building class: - build(config) - Build a wheel - build_all_platforms() - Build for all platforms

pyz3.wheel.build_wheel()

Convenience function for simple builds.
